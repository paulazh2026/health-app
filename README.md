# 个人健康管理应用

一个简洁美观的 PWA 健康管理应用，支持体重管理、饮食记录和营养分析。

## 功能特点

- 📊 仪表盘 - 今日健康数据概览
- ⚖️ 体重管理 - 记录体重变化趋势，BMI 计算
- 🍽️ 饮食记录 - 记录三餐，支持食物搜索
- 📈 营养分析 - 可视化营养摄入图表
- 📷 AI 识别 - 拍照识别食物（TensorFlow.js）
- 🌙 深色模式 - 保护眼睛
- 📱 PWA 支持 - 可添加到手机主屏幕

## 技术栈

- 纯 HTML/CSS/JavaScript
- IndexedDB 本地存储
- Chart.js 数据可视化
- TensorFlow.js AI 识别
